# Client

## Getting Started
1. Install Node.js & pnpm
    - For Node.js, nvs is recommended: https://github.com/jasongin/nvs
    - `npm i -g pnpm`
2. Run `pnpm dev`

## VS Code Extensions (Recommended)
- Error Lens --> Highly recommended
- Biome --> Highly recommended
- Path Intellisense
- Better Comments
- Code Spell Checker
